# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-15

### Added

- Initial release of Gloord Admin Dashboard
- Multi-language support (English, French, Arabic)
- Admin and Doctor role-based access control
- Patient management system
- Case review and approval workflow
- Analytics dashboard with charts and metrics
- Notification system
- Product catalog management
- Skin analysis and treatment planning
- Responsive design with Tailwind CSS

### Changed

- Migrated from hardcoded colors to theme tokens
- Added loading states for route segments
- Expanded translation coverage
- Updated test suite to include Safari and Firefox browsers
- Removed unused dependencies (Three.js, GSAP, Framer Motion)

### Fixed

- Various UI/UX improvements
- Performance optimizations
